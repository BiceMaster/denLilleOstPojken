The installation assumes that you can run the game in 1920x1080 fullscreen. 
However if you have a resultion lower than that you can manually change it
to your resolution by editing the following file "settings.cfg" Located at
the root of the installation path. 

Default path: "C:\Program Files (x86)\Majestic 6\The Little Cheese Boy"

Please also note that the later levels aren't the necessary the hardest ones!
